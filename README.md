


## 安裝與執行

```bash
# 安裝環境
npm install gulp -g

# Clone 專案
git clone git@github.com:ayugioh2003/Sweataste4.0.git

# 執行
npm install
gulp

# 其他
gulp build # 生成靜態網頁（未壓縮）
gulp build --env pro # 生成靜態網頁（壓縮）
gulp build --env production
gulp buildPro
gulp deploy # 將靜態網頁佈署到 gh-pages
```
